# DirectX_MyEngine  

## 開発環境
DirectX12
windowsSDK ver “10.0.22000.0"

## 使用言語

### C++

## 特徴  

・2D画像3Dオブジェクトの表示、サウンド再生など最低限のゲーム基盤を作成出来ます。  
  
・IMGUIを使った実行中においてのデバッグが可能 
  
・Blenderを用いて、ゲーム内オブジェクトの配置が可能
